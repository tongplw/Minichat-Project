# Minichat Project

## Getting Started

### 1 Development

1. Install libraries
``` Shell
cd <root>/minichat
pip install -r requirements.txt
```

2. run development mode
``` Shell
cd <root>/minichat
flask run 
```
Then, the development website can be accessed in http://localhost:5000/
