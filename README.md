# Minichat Project

## Getting Started

### 1 Development

``` Shell
cd <root>/minichat
python main.py
```
Then, the development website can be accessed in http://localhost:5000/

### 2 Deployment

To run development mode, go into /.../minichat/ and run 
```
flask run 
```

`Master` branch is shown in http://chat.tongplw.codes/
